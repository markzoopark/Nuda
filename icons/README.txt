Для завершения расширения необходимо добавить три иконки в эту папку:

1. icon16.png - размер 16x16 пикселей
2. icon48.png - размер 48x48 пикселей 
3. icon128.png - размер 128x128 пикселей

Рекомендуется создать простые иконки с буквой "F" (от Font) в квадрате с закругленными углами.
Можно использовать любой графический редактор или онлайн-сервис для создания иконок.

Примечание: Chrome Web Store требует качественные иконки, соответствующие рекомендациям:
https://developer.chrome.com/docs/webstore/images
